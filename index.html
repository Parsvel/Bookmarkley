Features

Tab Renamer

Renames the active Tab's title to a custom input.

Takeover

Completley takesover the website and installs a script to change the website content onload.

Browser/Proxy

Uses our proxy service to give you a unblocked browser experience in your Tab.

How does Bookmarkley work?

Bookmarkley is a bookmarklet. A Bookmarklet is a bookmark with a string of javascript instead of a link to a website, so when you click the bookmark, it injects that javascript into the webpage and exectues it.
